This is a reference dApp store implementaion for Meroku DAO.

Steps to run :

1. Set up env as per .env file.
2. Run `yarn`
3. Run `yarn run dev`

Steps to build :

1. Set up env as per .env file.
2. Run `yarn`
3. RUN `yarn build`
4. Start the next server with `yarn start`
